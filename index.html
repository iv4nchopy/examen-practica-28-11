Descripción del Sitio

Este proyecto integra todos los contenidos trabajados durante el curso: estructuras HTML, estilos con CSS, interacciones con JavaScript, formularios, consumo de APIs, multimedia, simulación de red social y distintos tipos de menús.

  • HTML Básico: estructura base
  • Banco Biz: tablas de datos
  • Galería: manejo de imágenes
  • Operaciones: cálculos con JavaScript
  • Formularios: matriculación y examen
  • APIs: clima, países, Pokémon, Rick & Morty y criptomonedas
  • Multimedia: mapas, YouTube, videos y audios
  • Red Social: publicaciones con likes
  • Menús Personalizados: 3 estilos diferentes
