¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear y estructurar páginas web. Define la estructura del contenido como títulos, párrafos, enlaces, imágenes, listas, tablas y formularios.

Estructura básica de un documento HTML

<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>Título</title>
  </head>
  <body>
    Contenido de la página
  </body>
</html>

Etiquetas más utilizadas

  • <h1> a <h6> → Títulos
  • <p> → Párrafos
  • <a> → Enlaces
  • <img> → Imágenes
  • <ul> / <ol> → Listas
  • <table> → Tablas
  • <form> → Formularios
